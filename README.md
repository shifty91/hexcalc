# HexCalc #

## About ##

`hexcalc` is a hex calculator.

## Examples ##

### Interactive ###

    $ hexcalc
    => 0x17 + 23
    46 : 0x2e
    =>

### Non-Interactive ###

    $ echo '0x17 + 23' | hexcalc
    46 : 0x2e

## Build ##

    $ cargo install --path .
